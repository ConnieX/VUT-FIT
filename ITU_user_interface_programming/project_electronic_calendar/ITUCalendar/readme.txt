Projekt je vytvořený v WPF, pro kompilaci projektu je potřebné Visual Studio 2019 (použitá verze 16.3.10)
Visual Studio musí mít nainstalovaný balíček pro vývoj WPF aplikací
Projekt se otevírá otevřením souboru ITUCalendar.sln ve Visual Studiu
Jediná externí knihovna je NewtonSoft.Json 12.0.3 kterou by si mělo Visual Studio automaticky stáhnout po otevření
Kdyby se tak nestalo a došlo k chybě při kompilaci, projekt se stále načítá v pozadí a balíčky se teprve budou stahovat
